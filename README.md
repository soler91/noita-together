# noita-together
Play alone together

## current features
* Shared hearts
* Shared orbs
* Send spells
* Send gold
* Sync gods mood
* Sync death
* Send wands
* Send flasks
* [More details](https://github.com/soler91/noita-together/wiki/Features)
